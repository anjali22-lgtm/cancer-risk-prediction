# 🧬 Cancer Risk Prediction using Machine Learning

## 📌 Project Overview
This project predicts cancer risk levels (Low, Medium, High) using machine learning techniques based on demographic and health-related features. The goal is to assist in early risk detection and awareness.

---

## 📊 Dataset
- Source: Kaggle Cancer Risk Factors Dataset
- Features include:
  - Demographic data
  - Health indicators
  - Behavioral factors

---

## ⚙️ Workflow

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Handling Missing Values
4. Label Encoding (categorical → numerical)
5. Feature Scaling (Standardization)
6. Train-Test Split
7. Handling Class Imbalance using SMOTE
8. Model Training
9. Model Evaluation

---

## 🤖 Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier

---

## 🏆 Results

| Model                | Accuracy |
|---------------------|----------|
| Logistic Regression | ~99%     |
| Random Forest       | ~99%     |

✔ Best Model: Random Forest Classifier  
✔ Cross Validation Score: ~0.999

## 📈 Techniques Used

- Pandas & NumPy for data handling
- Matplotlib & Seaborn for visualization
- Scikit-learn for ML models
- SMOTE for handling class imbalance
